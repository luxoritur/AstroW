Botones
Ver código fuente

Botones clásicos, en diferentes, colores, tamaños y estilos

Clases para crear botones
  • btn: Botón base sin fondo, Ideal para mantener un estilo minimalista.
  • btn-{color}: Botón con fondo sólido basado en el color especificado.
  • btn-outlined-{color}: Botón con fondo blanco y borde coloreado.
  • btn-complement-{color}: Botón con estilo complementario: el color de fondo lo definís vos, y al hacer hover, se intercambian los colores del texto y el fondo.

<button class="btn br-lg">Boton</button>

<button class="btn-info br-lg">Boton</button>

<button class="btn-outlined-info br-lg ">Boton</button>

<button class="btn-complement-link br-lg">Boton</button>
Sizes

Variaciones de tamanos para los diferentes tipos de botones


Clases para crear botones con las diferentes variaciones de tamaño.
  • btn-{size}
  • btn-{color}-{size}
  • btn-outlined-{color}-{size}
  • btn-complement-{color}-{size} 

<button class="btn-sm br-lg">Boton</button>

<button class="btn-info-md br-lg">Boton</button>

<button class="btn-outlined-info-lg br-lg ">Boton</button>

<button class="btn-complement-link-xl br-lg">Boton</button>

Badges

Ver código fuente
  • badge-solid-{color}: Badge con fondo solido y texto
  • badge-outlined-{color}: Badge con borde de color y fondo transparente
  • badge-soft-{color}: Badge con un fondo claro (tono suave del color)
  • badge-inverted-{color}: Badge con fondo claro y texto del color base. Ideal para fondos oscuros
