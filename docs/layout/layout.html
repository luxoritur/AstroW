Ejemplo HTML 

<div class="grid xl:grid-cols-12 lg:grid-cols-6 md:grid-cols-3 xs:grid-cols-2">

</div>

Comportamiento

Cuando la pantalla tiene 250px o más (xs): el grid muestra 2 columnas.

A partir de 640px (md): el grid cambia a 3 columnas.

A partir de 960px (lg): el grid muestra 6 columnas.

A partir de 1200px (xl): el grid muestra el máximo de 12 columnas.
